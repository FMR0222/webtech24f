Picture Gallery with JavaScript

The page below holds a table that has 2 rows and 2 columns, each cell holding an image. Think of this as your thumbnail view. The table has been floated left. There is another image to the right of the table, that is currently empty and hence you cannot see it. It has an ID of "big" and we will refer to this as the big image. Below the big image is a caption, also empty, and invisible. Write the functionality in Javascript whereby, if you click on a thumbnail image, that image is shown as a big image, much like how a picture gallery will work. Prompt the user for a caption each time and show the caption below the image using the placeholder provided.
